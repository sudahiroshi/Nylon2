まだ作っている途中ですが，新しいnylonです．これまでのnylonで問題となっていた，自分自身にもイベントが飛んで来る問題を解消しました．ここから飛べるgithubに，Webクライアント版のEventEmitterがある．元々node.jsの基幹部として作られたEventEmitterの移植版の一つで，本研究室とは無関係．同じくEventEmitterの移植の一つである石井くんのコードと合体させて，nylon.client.emitする形から<a object>.emitする形に変更．これにより，自分に対してメッセージを送らないように改造を施せました．ちゃんとしたコーディングマニュアルはそのうち作るとして，まずは簡単な使い方です．※現状ではCOeを始め，全てのライブラリが動作しません．

使い方　その1　{}に対するイベント登録方法


var object1 = {};
nylon.extends( object1 );
object1.addEventListener( "sample", function(e) {
	console.log( e + " by object1" );
});

使い方　その2　functionに対するイベント登録方法


function func(){};
func.prototype = new nylon();
var object2 = new func();
object2.addEventListener( "sample", function(e, arg) {
	console.log( e + " by object2 " + arg );
});

使い方　その3　イベント呼び出し方法


object1.emit( "sample" );	// -> object2のイベントのみ反応
object2.emit( "sample", "message" ); // -> object1のイベントのみ反応
